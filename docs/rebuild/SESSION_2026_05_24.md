- Every new function MUST have a manifest. The schema is fixed (FunctionManifest pydantic + matching TS).

- Every new function MUST honor the DataMode enum and report sources + as_of + latency.

The rebuild is now an incremental walk along a fixed contract, not an open-ended rewrite.
