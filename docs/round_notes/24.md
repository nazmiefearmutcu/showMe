## Decisions / deviations

* **CN alias kayıtta, NI ana isim.** ShowMe 138-fn katalog'unda
  `CN` Bloomberg'in tarihsel kısaltması, `NI` ise news-item
  default. Sidecar hangisini export ederse export etsin, registry
  iki yönde de UI çözer; runtime fetch tarafı `404 → CN` retry ile
  korunuyor.
* **MOST sort segmented control, header trailing'de.** Volume/$ vol/
  abs change üçü trader için "ilk bakışta" dimensiyonlar; üçünü de
  client-side sort ile sunmak server-side parametre genişlemesini
  şimdilik ertelyor.
* **WEI region client-fallback.** Bazı ShowMe builds region parametresine
  cevap vermiyor; payload region alanı varsa local filter, yoksa "All"
  görünümü.
* **HP range max = 25 yıl.** Bloomberg HP "max" ≅ 30y; ShowMe
  yfinance/Binance datas'ında 25y pratik tavan, infinite scroll yerine
  fix range tab daha az UX sürpriz.
* **CSV in-memory Blob, sidecar export endpoint yok.** ShowMe zaten
  CSV export'unu desteklemiyor; client-side Blob bridge ile beklenmeyen
  Tauri permission'a gerek kalmadı (`save dialog`'a Round 28 auto-update
  paketinde geçeriz, sırası gelir).
