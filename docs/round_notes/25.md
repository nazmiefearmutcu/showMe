# Round 25 — TRAN / WCRS / GLCO panes + builtin Markets Overview presets

**Date:** 2026-05-01
**Goal (direktif §0):** Üçüncü pane dalgası — trade blotter (TRAN),
FX cross matrix (WCRS), commodities mini-board (GLCO) — ve splash
ekranından tek tıkla yüklenen 3 builtin workspace preset (Markets
Overview, Trading Desk, Macro Watch). Native pane sayısını N=19'a
çıkarıyor; direktif §0 hedefi (Round 25 sonu N≥15) güvenli aralıkta.

## Delivered

### Sidecar — `state_api.py` + 3 yeni route

* `list_positions(db)` — `unrealized_pnl DESC, imported_at DESC`
  sıralı; `raw_json` blob'unu `raw` dict'ine hidrate ediyor.
* `list_trades(db, limit, symbol)` — closed-first, opsiyonel symbol
  filter (büyük harfe çevrilir), total count ayrı sorguda.
* `list_migrations(db, limit)` — audit trail; `summary_json` →
  `summary` decode.
* DB yoksa boş `StateRead` döner (FileNotFound exception fırlatmaz)
  — ilk açılışta migration çalıştırılmamış kullanıcı için fail-soft.
* FastAPI: `/api/state/positions`, `/api/state/trades`,
  `/api/state/migrations` — server.py içine `/api/function-index`
  hattı yanına gömüldü.

### TRAN pane (`functions/TRAN.tsx`)

* `lib/state.ts → listTrades({limit, symbol})` çağrısı; 60 sn
  auto-refresh + manuel `⟳`.
* Symbol filter input + limit input toolbar; symbol input büyük
  harfe çevirilir.
* DataGrid kolonları: closed_at, symbol (DES jump), side pill
  (positive/negative), qty, entry, exit, realized_pnl
  (ChangeText, prefix "$"), mode pill (writable warn).
* Header pill cluster: closed count, wins, losses, realized $.
* `buildTradeCsv()` + Blob URL download (HP.tsx pattern reuse).

### WCRS pane (`functions/WCRS.tsx`)

* `useFunction("WCRS", {base, tick})` 30 sn auto-refresh.
* Base segmented (USD/EUR/GBP/JPY/TRY) → backend filtre + client
  filter (base ya da quote eşleşmesi).
* `normalizeRows()` matrix payload'ı (`{matrix: {USD: {EUR: 0.93}}}`)
  flat array'a çeviriyor; flat array de kabul ediliyor.
* Kolonlar: pair, rate, bid, ask, spread (pips, `(ask-bid)*10000`),
  Δ %.
* `fmtRate()` JPY-style heuristic: |v|>20 → 2 dp, aksi 4 dp.

### GLCO pane (`functions/GLCO.tsx`)

* `useFunction("GLCO", {sector, tick})` 60 sn auto-refresh.
* Sector segmented: All / Energy / Metals / Ag / Softs.
* `matchesSector()` client fallback: "ags" → grain/ag substring,
  "softs" → soft substring, diğerleri direkt substring match.
* Kolonlar: symbol (DES jump), name, sector pill, last (+ unit
  inline), Δ %, contract month, open interest (compact).

### Built-in presets (`lib/builtinPresets.ts`)

* `BUILTIN_PRESETS` — 3 entry: **Markets Overview** (DES + GP üstte,
  WEI + TOP altta, 55/45 vertical split), **Trading Desk** (DES +
  PORT solda, GP + WATCH sağda), **Macro Watch** (WEI + WCRS üstte,
  GLCO + ECO altta).
* `loadBuiltinPreset(id, symbol?)` — `loadWorkspace()`
  serialize→deserialize döngüsünden geçer; id'leri remap eder, focus
  default ilk leaf.
* `panes/Welcome.tsx` — yeni "Workspace presets" Card; her preset için
  label + description + "Open ↗" butonu, click sonrası uygun
  fonksiyona navigate ediyor (Markets/Trading → DES, Macro → WEI).

### Registry (`functions/registry.tsx`)

* Yeni anahtar: `TRAN`, `WCRS`, `GLCO`.
* `listNativeCodes()` → 20 entry, 19 unique pane (NI/CN aliased).

## Decisions / deviations

* **State API ayrı modül, function endpoint ile karışmadı.** TRAN
  iş yükü ShowMe fonksiyonu değil — Faz B importer çıktısı. `/api/fn/`
  yolu üzerinden zorlamak yerine `/api/state/` ön-eki ile semantik
  ayrım korundu. Round 27 broker engine de `/api/state/orders` üzerinden
  aynı patterni izleyecek.
* **TRAN auto-refresh 60 sn.** Trade blotter price stream değil; 60
  sn aralık portfolio.db'nin yeniden import edilmesi senaryosunu tutar.
* **Builtin preset 3 adet, daha fazla değil.** Preset patlaması command
  palette'i kalabalıklaştırır. Round 26+ trader feedback'ine göre
  genişletilecek.
* **Welcome screen "Quick launch" altında değil üstünde gösteriliyor.**
  Trader app'i açar açmaz "Markets Overview"a tek tıkla girebilsin —
  function inventory aşağıda kalır.
* **`raw_json` hidratasyonu opsiyonel.** SQLite blob'u JSON olarak
  parse edilemezse string olarak korunur. Round 22 importer her zaman
  geçerli JSON üretiyor ama defensive.
