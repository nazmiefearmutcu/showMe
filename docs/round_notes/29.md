# Round 29 — Real-time WS stream (Binance bridge + polling fallback)

**Date:** 2026-05-01
**Goal (direktif §0):** Sub-second tick'ler trader'ın WATCH/PORT/MOST
yüzeyinde — Binance WS köprüsü kripto için, equity / FX için ShowMe
DES polling fallback. Hem real-time push hem de ShowMe'ün mevcut
adapter zincirini tek yerden kullanan unified stream contract.

## Delivered

### Sidecar — `src-py/showme/streams.py`

* **`Tick`** dataclass — symbol, price, change_pct, volume, bid, ask,
  ts (epoch sec), source.
* **`Source` ABC** — async iterator + cooperative `stop()`.
* **`BinanceWsSource`** — `wss://stream.binance.com:9443/ws/<symbol>@ticker`;
  `transport: Callable[[str], WsConn]` enjekte edilebilir (production
  build `websockets` paketini içeriye lazy import edecek).
* **`parse_binance_ticker()`** — pure-Python; `s/c/P/v/b/a/E` →
  `Tick`. Eksik field'lara karşı dirençli (`None` döner, hata
  fırlatmaz).
* **`is_crypto_symbol(symbol)`** — `*USDT / *USD / *BTC` → True;
  `AAPL` → False.
* **`PollingSource`** — `fetch: (symbol) -> Awaitable[dict]` enjekte
  edilebilir; default `interval=5.0s`; payload `{regularMarketPrice,
  previousClose}` veya FunctionResult envelope kabul eder.
* **`StreamHub`** — per-symbol fan-out; ilk subscribe `Source` ile
  başlar, `last_tick` cache ile yeni subscriber instant paint alır,
  son subscriber ayrılınca source `stop()`'lanır + asyncio Task
  iptal edilir. `Subscription` async context manager — `async with`
  ile temiz tear-down.
* **`StreamHub.stats()`** — debug endpoint için kanal listesi (symbol,
  subscriber count, last price, source).

### Sidecar — `server.py` route'ları

* `GET /api/stream/stats` — hub durumu.
* `WS /ws/quote/{symbol}` — fan-out subscribe; `Tick.to_dict()`
  JSON push; subscriber drop'unda hub release. Hata durumunda
  `{"error": ...}` mesaj + close.
* Lazy hub init — ShowMe attach yoksa polling fetch'i boş döner;
  websockets bağımlılığı sadece prod transport'ta yüklenir.

### Frontend — `lib/sidecar.ts` + `lib/stream.ts`

* `sidecarWsUrl()` helper — sidecar port keşfedilince
  `ws://127.0.0.1:<port>` döner.
* `subscribeQuote(symbol, opts)` — WebSocket factory; `onTick` +
  `onStatus` callback'leri; auto-reconnect exponential backoff
  (capped 5 s); status: `connecting / live / offline / error`.
* Returns `{close: () => void}` — caller unmount'ta çağırır.
* `signal: AbortSignal` opsiyonel; abort olursa socket kapanır.

### WATCH pane — push entegrasyonu

* `useEffect(rows)` her satır için `subscribeQuote()`; `onTick` row
  state'ini günceller, `onStatus` per-row stream durumu izler.
* Yeni "Stream" kolonu Pill ile renklendirilir
  (`live=positive`, `offline=muted`, `error=negative`,
  `connecting/warn`).
* Footer'da `"ws · X/N live"` özet sayacı.
* 30 sn DES polling polling fallback olarak korunur — transient WS
  hata durumunda WATCH boş kalmaz.

## Decisions / deviations

* **`websockets` paketi dev'de kurulu değil** — `_default_ws_transport`
  açık şekilde RuntimeError; testler injectable transport'la geçer.
  CI'da prod build'de `pip install websockets` adımı eklenmiyor;
  Round 30+ packaging adımında pin'lenecek.
* **Source'tan tick `last_tick` cache'i hub seviyesinde tutulur** —
  yeni subscriber instant paint alır. Backpressure: queue full →
  oldest drop (subscriber yavaşsa son tick öncelikli).
* **Polling fallback default 5 sn** — ShowMe yfinance rate limit'leri
  daha agresif değerleri kaldırmaz; user override için interval
  parametresi kabul ediliyor.
* **WATCH 30 sn DES polling korundu** — WS push live olduğu sürece
  güncellemeler push'tan, kesintide DES polling devreye girer (belt-
  and-suspenders).
* **`is_crypto_symbol` Binance konvansiyonuna göre** — `BTCBTC`
  veya `*BTC` pair'lar yine kripto sayılır; `is_*` heuristic'inden
  yanılgıya düşersek hub `runtime error` ile koruma sağlar.
* **Hub fan-out asyncio.Queue maxsize=128** — yavaş tüketici için
  buffer; aşılırsa drop-oldest.
