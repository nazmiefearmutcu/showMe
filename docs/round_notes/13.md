## Decisions / deviations

* **Hash router over React-Router.** `react-router-dom@6` would add 60 KB +
  client-side history quirks the Tauri webview doesn't need. The hash
  approach also keeps deep-link forwarding from Round 12 trivial — Rust
  emits `nav:open` with a path, JS calls `navigate(payload)` and
  `window.location.hash` updates.
* **Theme toggle local-only for now.** Round 16+ subscribes to OS
  appearance via `tauri-plugin-os` and offers an "auto" option.
* **i18n: dropped runtime-fetched catalogs.** Catalogs are imported as
  modules so Vite tree-shakes them and there's zero waterfall on first
  paint. When we add 10 more languages they each cost <1 KB gzipped.
* **`Pill` superseded the inline `.pill` CSS class** that Round 12
  shipped in `index.css`. The CSS rule is left in place because the
  shell's status indicator is rendered via `Pill` now anyway —
  redundancy is fine; cleanup deferred to Round 14.
* **Vitest stub for localStorage.** Documented in `src/test/setup.ts`
  comment so future contributors don't try to "fix" it.
* **Sidebar buttons are `<button>` not `<div>`.** A11y matters; it also
  gives free `:focus-visible` rings via tokens.css.
