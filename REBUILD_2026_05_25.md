## Öncesi vs sonrası

| Yüzey | Öncesi | Sonrası |
|---|---|---|
| Fonksiyon contract | İmplicit — her panel kendi envelope'ını uyduruyordu | Explicit `FunctionManifest` (pydantic v2 + TypeScript), 143 kod için kayıtlı, `/api/manifest/{code}` ile servis ediliyor |
| Data honesty | `status="ok"` synthetic / placeholder payload'da bile | `DataMode` enum: `live_official` / `live_exchange` / `delayed_reference` / `modeled` / `cached_snapshot` / `provider_unavailable` / `not_configured`. Her panel header'ında pill olarak görünür. |
| Production fakery | Workspace.tsx'de FunctionStub + TemplateRenderer JSX; design-export production path'inde; templates routing | **0 violation** (strict-zero gate `tests/test_production_fakery_banned.py::test_production_fakery_strict_zero` artık AÇIK). Workspace fallback `<ManifestPane>`'e gidiyor. `/api/templates/*` `SHOWME_DEV=1` arkasına gated. |
| Provider katmanı | Handler'lar içinde dağınık `httpx` / `requests` çağrıları, ad-hoc cache | 8 adapter tek ABC arkasında: SEC EDGAR, FRED, TreasuryDirect, OpenFIGI, Binance, yfinance, GDELT, RSS. Shared `httpx.AsyncClient`. `auth_state()`, `last_latency_ms()`, `mode()` API'leri. Lifespan startup'ta register. |
| Local cache | Dağınık SQLite + `functools.lru_cache` | `backend/showme/analytical/` — tek DuckDB DB (`<state>/analytical.duckdb`), schema'lar: `cache.*`, `snapshot.*`, `audit.*`, `research.*`. Polars helpers (resample, join_asof, correlation_matrix...). |
| Chart grammar | Birçok panel "row index vs metric" plot — yanlış (correlation için heatmap, GEX için strike ladder, CRVF için tenor curve gerekli) | Manifest'in `chart_grammar.kind` field'ı kontrat: CORR=heatmap, PORT_OPT=frontier, GEX=bar_ladder, IVOL/OVDV=surface, CRVF=tenor_curve, GC3D=surface, WCRS/MAP/FRH=heatmap, WEI/MOST/MOSS/SECT=bar_ladder, OVME/OSA=payoff. |
| Sentiment | FinBERT model bundle'da vardı ama hiçbir handler çağırmıyordu | `backend/showme/finbert_analyzer.py` singleton; TOP/NI/CN/INSTANT her item'a `sentiment` + `sentiment_score` damgalıyor; lifespan warmup → `/api/health.engine.finbert_warmed`. |
| Transcripts | Whisper bundle'da ama default `"base"` ve hiçbir handler kullanmıyor | `backend/showme/whisper_analyzer.py` singleton, `openai/whisper-large-v3`'e pinned; TRAN/TRQA "whisper warming" warning'i `provider_errors`'ta; TSAR `ingest` action auto-Whisper + auto-FinBert sentiment; `boot_state["whisper_warmed"]` `/api/health`'te. |
| WIRP | `cme_fedwatch` adapter loaded ama `wirp.py` handler tüketmiyordu — implied path deterministic fallback'ten geliyordu | Handler `self.deps.cme_fedwatch`'i FED bank için tüketiyor; `data_mode="live_official"` + `provenance.sources=["cme_fedwatch"]`; CME yokken `modeled` mode + warning. cut+hold+hike=1.0 invariant korundu. |
| Pane görünür kontrat | Yoktu — her panel kendi header tasarımıyla | PaneChrome 26px row'una inline `<ContractInline>` eklendi: `📜` manifest dot (tooltip: category + primary provider + N inputs) + data-mode pill + first source + ⚠N warnings count. `useFunction` ve `sidecarFetch` her `/api/fn/{code}` cevabını otomatik olarak `pane-contract-store`'a yazıyor; bespoke panel'ler (GP/HP/PORT/DES/WATCH/...) refactor gerektirmeden strip aldı. |
